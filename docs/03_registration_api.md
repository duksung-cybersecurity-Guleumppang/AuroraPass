# API 명세: 03. 수강 신청 시 CAPTCHA 인증

이 문서에서는 수강 신청 과정에서 비정상적인 접근이 탐지되었을 때 사용되는 CAPTCHA 인증 API를 설명합니다.

| API 기능 | HTTP | API Path | 설명 |
| --- | --- | --- | --- |
| **수강신청 요청** | `POST` | `/api/enroll` | 수강신청을 시도합니다. 비정상적인 접근으로 판단될 경우, CAPTCHA 인증을 요구하는 응답을 반환합니다. |
| **CAPTCHA 생성 요청** | `GET` | `/api/captcha` | 비정상 접근이 탐지된 사용자에게 CAPTCHA 문제(오디오/이미지)를 제공합니다. |
| **CAPTCHA 정답 제출 및 검증** | `POST` | `/api/captcha/verify` | 사용자가 입력한 CAPTCHA 정답을 검증합니다. 성공 시, 서버는 해당 사용자의 요청을 신뢰할 수 있다고 판단하는 인증 플래그를 부여합니다. |
