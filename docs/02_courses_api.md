# API 명세: 02. 강의 관리 및 수강 신청

이 문서에서는 강의 조회, 장바구니, 수강 신청 등 핵심 기능 관련 API를 설명합니다.

| API 기능 | HTTP | API Path | 설명 |
| --- | --- | --- | --- |
| **강의 목록 조회** | `GET` | `/api/courses` | 개설된 전체 강의 목록을 조회합니다 (과목명, 교수명, 시간, 정원 포함). |
| **장바구니에 강의 추가** | `POST` | `/api/cart` | 선택한 과목을 장바구니에 추가합니다. 중복 과목 추가 방지 처리가 포함됩니다. |
| **장바구니 목록 조회** | `GET` | `/api/cart` | 사용자가 장바구니에 담은 과목 목록을 조회합니다. |
| **장바구니에서 과목 제거** | `DELETE` | `/api/cart/{courseId}` | 장바구니에서 특정 과목을 제거합니다. |
| **수강신청 (본 신청)** | `POST` | `/api/enroll` | 장바구니에서 선택한 과목들을 최종적으로 신청합니다. 정원 초과, 시간표 중복, CAPTCHA 대응 로직이 포함됩니다. |
| **수강신청 결과 조회** | `GET` | `/api/my-courses` | 신청에 성공한 과목 목록을 확인합니다. 시간표 또는 마이페이지에서 사용됩니다. |
